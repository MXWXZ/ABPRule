# 规则补充
本规则仅对默认和已有规则未完善之处给予补充\
以下列表内规则不在本规则内：
- EasyList
- EasyList China
- CJX's Annoyance List
- ABP filters
- EasyPrivacy

# 规则说明
1. 本规则将会去除广告占用页面的所有元素，不会造成视觉缺陷
2. 所有含推广性质的内容均会被本规则过滤，没有例外
